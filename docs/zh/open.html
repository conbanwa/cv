moe-data.github.io https://moe-data.github.io long term
简介: Github 开源项目。一个网页游戏数据分析器。
本站的主要功能是多装备/多舰娘开发查询和数据解析展示。
由于游戏官方没有公开配方与建造结果之间的对应关系，Moe-data 收集玩家在游戏中使用的所有配方与建造结果。
服务器根据多舰娘建造/多装备开发计算每种建造结果的概率，并响应网站的各类查询。
语言、框架: HTML5、MySQL、MongoDB、JavaScript、jQuery、echarts。
